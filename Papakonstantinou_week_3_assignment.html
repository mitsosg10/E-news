Marks: 60

Define Problem Statement and Objectives¶

The number of new members to the online news portal E-news Express is declining each month. The business feels that the content and layout of the website as it is now isn't compelling enough to get users to subscribe. The design team has produced a new landing page in order to resolve this problem. Determining if this new landing page is more successful at drawing visitors in and turning them into subscribers is the current challenge. Two primary metricsthe duration of time users spend on the website and the conversion rate, or the percentage of visitors who become subscribers, will be used to assess how effective the new landing page is. The particular queries that require responses are:

Do users spend more time on the new landing page than on the existing landing page?

Is the conversion rate for the new page greater than the conversion rate for the old page?

Does the converted status depend on the preferred language of the user?

Is the time spent on the new page the same for users who prefer different languages?

The answers to these questions will help E-news Express make an informed decision about whether to implement the new landing page.

1. Do the users spend more time on the new landing page than the existing landing page?¶

Perform Visual Analysis¶

Step 1: Define the null and alternate hypotheses¶

H0 : Users does not spend more time on the new landing page than the existing landing page.

H1 : Users spend more time on the new landing page than the existing landing page.

Step 2: Select Appropriate test¶

We need a one-tailed test concerning two population means from two independent populations where the population standard deviations are unknown. The appropriate test is the two-sample t-test for independent samples with unequal variances, known as Whelch's test.

Step 3: Decide the significance level¶

The required level of significanse is a=0.05 and will be used on every test follows.

Step 4: Collect and prepare data¶

In [ ]:
# We need a subset of time spent on the new and the old page.
# new
time_new =  df[df['landing_page'] == 'new']['time_spent_on_the_page']
# old
time_old = df[df['landing_page'] == 'old']['time_spent_on_the_page']

print('The standard deviation of the time spent on the new page is:', round(time_new.std(),2))
print('The standard deviation of the time spent on the old page is:', round(time_old.std(),2))

The sample standard deviation of the time spent on the new page is: 1.82
The sample standard deviation of the time spent on the old page is: 2.58

Variances are not equal.

Step 5: Calculate the p-value¶

In [ ]:
from scipy import stats


# Perform Welch's t-test
t_stat, p_value = stats.ttest_ind(time_new, time_old, equal_var=False,alternative='greater')

print('p_value is',p_value)

p_value is 0.0001392381225166549

Step 6: Compare the p-value with $\alpha$¶

In [ ]:
# Check the p-value to determine significance
if p_value < 0.05:
    print("We have enough statistical evidence to reject the null hypothesis - There is a significant difference!")
else:
    print("We do not have enough statistical evidence to reject the null hypothesis - There is no significant difference.")

Reject the null hypothesis - there is a significant difference.

We have enough statistical evidence to say that users will spend more time on the new page.

Step 1: Define the null and alternate hypotheses¶

H0: There is no significant difference between the proportions of control and treatment groups.

Ha: There is a significant difference between the proportions of control and treatment groups, more users are converted to subscribers using the new page.

Step 2: Select Appropriate test¶

For a one-tailed test concerning two population proportions from two independent populations, the appropriate test is the two-sample z-test for proportions. This test is used to determine if there is a significant difference between the proportions of two independent groups.

For comparing three population means, the appropriate test is the Analysis of Variance (ANOVA). Specifically, if we are comparing means across three or more groups, we would use a one-way ANOVA.

Conclusion and Business Recommendations¶

Conclusions:¶

The new landing page seems to be more effective in terms of both user engagement and conversion rate. Users are spending more time on the new page and the conversion rate is also higher on the new page compared to the old one.¶

The preferred language of the user does not seem to have a significant impact on the conversion rate. This suggests that the content of the landing page is equally effective across different languages.¶

There is no significant difference in the time spent on the new page across different language users. This indicates that the new page is equally engaging for users of all languages.¶

Recommendations:¶

Considering the higher engagement and conversion rate on the new landing page, it is recommended to replace the old landing page with the new one.¶

Since the preferred language does not seem to affect the conversion rate, it is beneficial to continue providing content in all current languages to cater to a diverse user base.¶

Further analysis could be done to understand other factors that might affect user engagement and conversion rate. For example, the type of content that users interact with the most, the time of day when users are most active, etc. This could provide more insights to further improve the landing page and increase the conversion rate.¶
